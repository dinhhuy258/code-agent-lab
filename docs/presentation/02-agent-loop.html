Core Architecture

The ReAct Loop

Reasoning + Acting — the agent iterates between thinking and doing until the task is complete.

STEP 1
📥
Observe
Receive user message, collect environment context
→
STEP 2
🧠
Reason
Send full history + tools to LLM, get structured response
→
STEP 3
🛠️
Act
Execute tool calls, append results to history
→
STEP 4
🔁
Loop
Repeat until text-only response or max turns (25)
💡
The model controls the loop. The harness is the executor; the LLM decides when to call tools, which tools to call, and when the task is done.
Implementation

AgentClient.send()

The core loop from our codebase — surprisingly simple at its heart.

core/agent_client.py
async def send(self, user_text: str): self._session.append_user_message(user_text) for _ in range(MAX_TURNS): # Send history + tools to LLM result = await self._session .send_message_stream() # Yield text chunks for streaming for chunk in result.text_chunks: yield TextChunk(chunk) # No tool calls? We're done if not result.function_calls: yield TextResponse(result.text) return # Execute tools, loop back await self._process_tool_calls( result.function_calls )

🔄 The Loop

Up to 25 iterations. Each iteration: call LLM → check for tool calls → execute or return.

📝 History is Memory

The conversation history is the agent's memory. Every message, tool call, and result stays in the list.

⚠️ Errors are Messages

Tool failures return as ToolResult.error — the model sees the error and can adjust its approach.

Architecture

Four-Layer Stack

Each layer has a single responsibility and can evolve independently.

Layer 4 — Orchestrator
AgentClient
The agent loop. Decides when to stop.
Layer 3 — History Manager
ChatSession
Maintains conversation history. Appends messages.
Layer 2 — Turn
Turn
Single request/response cycle. Extracts tool calls.
Layer 1 — LLM Client
GeminiLLMClient
Talks to the API. Handles auth, errors, streaming.
🏗️
Why layers? New features just "fill in" existing layers. Adding streaming didn't require rewriting the agent loop. Adding sub-agents didn't change the tool system.
